# Java OOP Banking System – Menu Driven Application

## Overview
A console-based banking system developed in Java to demonstrate practical implementation of object-oriented programming principles. The application simulates core banking operations and manages multiple customer accounts through a structured, menu-driven interface.

---

## Key Features
- Creation and management of multiple bank accounts  
- Automatic unique account number generation  
- Deposit and withdrawal operations  
- Address update functionality  
- Transaction counting mechanism  
- Structured menu-driven interaction  

---

## Technical Implementation
The system is designed using fundamental OOP concepts and standard Java utilities:

- Classes and Objects  
- Constructor Overloading  
- Static Members for ID generation and transaction tracking  
- Arrays of Objects for account management  
- Conditional Logic and Iterative Control Structures  
- Input handling using the Scanner class  

---

## Project Structure
The application centers around the `BankAccount` class, which encapsulates account-related data and operations such as deposit, withdrawal, and account updates. Multiple account instances are stored and managed dynamically using an array-based approach.

---

## How to Run

1. Clone the repository:
   git clone https://github.com/Apekshaabhat/Java-OOP-Banking-System---Menu-Driven-Application.git

2. Navigate to the project directory:
   cd Java-OOP-Banking-System---Menu-Driven-Application

3. Compile the program:
   javac BankAccount.java

4. Run the program:
   java BankAccount

---

## Sample Execution Flow

The system presents a structured menu allowing users to:
- View account details  
- Perform deposit or withdrawal  
- Update address information  
- View total transaction count  
- Exit the application
